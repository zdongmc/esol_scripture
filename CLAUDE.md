# ESOL Multi-language Bible Scripture Handout Generator

An web application deployed on github page (or equivalent) that has a GUI that users can choose from a set of preset scriptures and preset languages, and obtain a PDF that contains multiple translations of this scripture

Example:
    - App is preloaded with 20 scriptures (bible verses), each with their 20 translations from common languge
    - GUI allows user to pick a scripture from a table, and then pick a set of languages via checkboxes
    - User hits GO
    - The app will select the translations for that script, and put together a Markdown file, and then convert it into a downloadable PDF
    - The Markdown/PDF can be previewed on the GUI
    - A DOWNLOAD button shows up - the user can hit to download the PDF (or Markdown)
